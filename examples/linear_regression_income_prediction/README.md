## Linear Regression Income Prediction - PySpark ML Project

#### Installation:
```python
pip install pyspark-cli
```

#### Run the project:
```python
pysparkcli run linear_regression_income_prediction
```

#### Note:
The input data for the project must be placed in src/inputData folder.
