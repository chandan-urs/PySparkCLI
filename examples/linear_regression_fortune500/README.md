# Linear Regression Fortune 500 - PySpark ML Project

#### Installation:
```python
pip install pyspark-cli
```

#### Run the project:
```python
pysparkcli run linear_regression_fortune500
```

#### Note:
The input data for the project must be placed in src/inputData folder.
